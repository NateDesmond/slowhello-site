# Slow Hello - Marketing Site

Static GitHub Pages site for **Slow Hello**, served at [slowhello.urbanalgorithm.com](https://slowhello.urbanalgorithm.com).

Slow Hello is an iOS app by Urban Algorithm LLC: a message to the future, recorded a little at a time. Say it a word at a time.

## Files
- `index.html` - home page (hero, how it works, features, about, support)
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `favicon.svg` - the "Little Heartful" mark
- `CNAME` - custom domain for GitHub Pages

Plain static HTML and CSS, no build step. To preview, open `index.html` in a browser.

## Notes
- App is currently in TestFlight. When live, swap the "Coming soon" hero badge for the App Store link: `https://apps.apple.com/app/6782214415` (an HTML comment in `index.html` marks the spot).
